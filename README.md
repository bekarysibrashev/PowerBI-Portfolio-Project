# PowerBI-Portfolio-Project
В этом репозитории я хочу показать скилы в работе с PowerBI.

# 📊 Анализ данных продаж и построение дашборда (Power BI)

## Описание проекта

Данный проект представляет собой полный цикл анализа данных:  
от загрузки и очистки сырых данных до построения интерактивного дашборда в Power BI.

Цель моего проекта получить бизнес-инсайты на основе данных о продажах и выявить ключевые тенденции.

---

# Цель проекта
## Цель проекта
### Цель проекта
- Провести анализ продаж
- Выявить ключевые KPI бизнеса
- Определить тренды и сезонность
- Найти наиболее прибыльные категории товаров и клиентов
- Построить интерактивный дашборд для принятия решений

---

### Датасет

Источник: CSV файл с данными о продажах

Содержит следующие поля:
- дата покупки
- идентификатор клиента
- товар
- категория
- количество
- сумма покупки
- регион (если есть)

---

## Этап 1: Очистка данных (Data Cleaning)

В процессе подготовки данных были выполнены следующие шаги:

- Удаление дубликатов
- Обработка пропущенных значений
- Приведение типов данных (даты, числа)
- Исправление некорректных значений
- Стандартизация названий колонок

Результат: чистый и готовый к анализу датасет

---

##  Этап 2: ETL процесс

### Extract (извлечение)
- Загрузка исходного CSV файла

### Transform (преобразование)
- Очистка данных
- Создание новых признаков:
  - общая выручка
  - месяц/год продажи
  - средний чек
- Группировка данных по категориям и регионам

### Load (загрузка)
- Подготовленные данные были загружены в Power BI для визуализации

---

##  Этап 3: Исследовательский анализ данных (EDA)

В ходе анализа были изучены:

- Общая динамика продаж
- Топ товаров по выручке
- Распределение продаж по регионам
- Поведение клиентов
- Сезонность продаж

---

##  Этап 4: Дашборд (Power BI)

Был создан интерактивный дашборд, включающий:

- KPI карточки:
  - Общая выручка
  - Количество заказов
  - Средний чек

- Визуализации:
  - Динамика продаж по времени
  - Топ товаров
  - Продажи по регионам
  - Категории товаров

- Фильтры (slicers) по:
  - дате
  - категории
  - региону

 Дашборд позволяет анализировать данные в динамике и принимать бизнес-решения

---

##  Ключевые инсайты

- Небольшое количество товаров формирует основную часть выручки
- Есть выраженная сезонность продаж
- Некоторые регионы показывают значительно более высокие результаты
- Поведение клиентов различается по категориям товаров
