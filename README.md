# PowerBI-Portfolio-Project
В этом репозитории я хочу показать скилы в работе с PowerBI.

# Анализ данных продаж и построение дашборда (Power BI)

## Описание проекта

Данный проект представляет собой полный цикл анализа данных:  
от загрузки и очистки сырых данных до построения интерактивного дашборда в Power BI.

---

## Цель проекта

Провести анализ данных, выявить ключевые закономерности и представить результаты в виде интерактивного дашборда Power BI.

---
## Датасет

Источник: CSV файл с данными о машинах. Данный собраны из открытого источника Kaggle:
https://www.kaggle.com/datasets/gagandeep16/car-sales

### Содержит следующие поля:
- Производитель(Марка)
- Модель
- Количество продаж
- Цена через год(после покупки)
- Тип кузова(тут все разделено на 2 большие группы: большие машины(внедорожники, джипы, минивены) и маленькие машины(седаны))
- Цена
- Обьем двигателя
- Лошадиные силы
- Размер колес
- Ширина кузова
- Длина кузова
- Груженый Вес
- Обьем бака
- Эффективность топлива
- Дата изготовления
- КПД

## Этап 1: Очистка данных (Data Cleaning)

В процессе подготовки данных были выполнены следующие шаги:

- Удаление дубликатов
- Обработка пропущенных значений
- Приведение типов данных (даты, числа)
- Исправление некорректных значений
- Стандартизация названий колонок
- ETL

  
Результат: чистый и готовый к анализу датасет

---


##  Этап 2: Исследовательский анализ данных (EDA)

В ходе анализа были изучены:

- Общая динамика продаж
- Топ производителей машин,
- Анализ Концентрации рынка,
- Распределение продаж по маркам и производителям
- Аномалии 
- Сезонность продаж

---

##  Этап 4: Дашборд (Power BI)

Был создан интерактивный дашборд, включающий:

- Визуализации:
  - Динамика продаж по времени
  - Топ товаров
  - Продажи по регионам
  - Категории товаров

- Фильтры (slicers) по:
  - дате
  - категории
  - региону

 Дашборд позволяет анализировать данные в динамике

##  Ключевые инсайты && Используемые технологии

- Анализ показал что высокая цена новго автомобился не гарантирует стабильную цену на вторичном рынке. Жестким контрастом в Премиум сегменте оказались PORSCHE И BMW, поскольку они сохраняют 91.7% и 89.4% свой цены на вторичном рынке, тогда как LINCOLN явялется главным аутсайдером потому что за первый же год теряет до 49% свой стоимости.
- Ford F-Series -Легендарный пикап. Это главный финансовый донор компании. Его штучные продажи превосходят суммарные продажи многих других брендов, вместе взятых.
